## Компиляция    
### Linux:   
1. Запустить файл compile.sh:   
    > $ sudo ./compile.sh
2. Запустить файл proga.exe:    
    > $ ./proga.exe
    
### Windows:
1. Запустить файл compile.bat   
2. Запустить файл proga.exe
